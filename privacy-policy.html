Important: This extension does not collect, store, or transmit any user data. All processing happens locally in your browser.

Data Collection

None. This extension does not collect any user data, including:

  • Personal information
  • Usage statistics
  • Text content you encode or decode
  • Browser history or browsing data
  • Device information

Data Processing

All Base64 encoding and decoding operations happen locally in your browser using standard JavaScript functions:

  • atob() - for decoding Base64 strings
  • btoa() - for encoding text to Base64

No data is sent to external servers or processed outside your device.

Data Storage

None. This extension does not store any data:

  • No local storage
  • No cookies
  • No user preferences
  • No history of encoded/decoded text

Third-Party Services

This extension does not use any third-party services, analytics, or external APIs.

Permissions

The extension requests the following permission:

  • activeTab: This permission allows the extension to access the current tab when you click the extension icon. It is used to potentially access text you have selected on a webpage for encoding/decoding. No data is collected or transmitted.

Updates to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected in the extension's store listing and this document.
